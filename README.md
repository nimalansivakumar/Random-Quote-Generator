# 📝Quote Machine

Quote Machine is a random quote generator based on the user selected category.
![Quote Machine](imgs\md-image.png)

# Features
-📑 Contains more than 7+ categories
-🤵🏼 Famous quotes with author names
-🖨️ Users can copy the quote
